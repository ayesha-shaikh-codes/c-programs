# 1. BMI Calculator (c)

Console-based BMI (Body Mass Index) Calculator written in C that helps users calculate and understand their BMI based on height and weight input.

## Features

 - Calculates BMI using height and weight
 - Displays BMI value
 - Categorizes BMI into:
 - Underweight
 - Normal weight
 - Overweight
 - Obese
 - Provides a brief explanation for each category
 - Validates user input (accepts only positive values)

## Concepts Used

 - C Programming Language
 - Functions
 - Conditional Statements
 - Input Validation
 - Modular Programming

## Technologies Used

 - C Programming Language
 - GCC Compiler
   
## How to Run

1. Compile the program:
gcc bmi.c -o bmi
2. Run the program:
./bmi


# 2. Student Management System (C)

A menu-driven Student Management System written in C that allows users to manage student records efficiently.

## Features
- Add student details
- Display all students
- Search student by roll number
- Update student marks
- Delete student records

## Concepts Used
- C Programming Language
- Structures
- Functions
- Pointers
- Arrays

## Technologies Used
- C Programming Language
- GCC Compiler

## How to Run
1. Compile the program:
   gcc student_management.c -o student
2. Run the program: ./student
